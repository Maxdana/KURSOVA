# EncryptionSystem

Курсова робота з програмування мовою C++.

## Функціонал програми

* Шифр Цезаря
* XOR-шифрування
* Меню взаємодії з користувачем
* Управління алгоритмами шифрування через класи

## Технології

* C++
* Visual Studio
* ООП

## Запуск

Відкрити файл `EncryptionSystem.sln` у Visual Studio та запустити проєкт.
